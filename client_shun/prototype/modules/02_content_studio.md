# Module 02 — Content Studio（SNS動画AI）

## 目的
しゅんの声・話し方・編集スタイル・テンションを学習し、SNS動画（TikTok / Instagram / YouTube）を量産。
撮影・編集に取られる週20時間を解放する。

## Voice Profiles（プロトタイプで定義済）
| プロファイル | 用途 | 特徴 |
|---|---|---|
| TikTok・煽り系 | 短尺バズ動画 | スピード感、効果音、爆笑系 |
| Instagram・上品系 | ストーリー/リール | 高画質、英字字幕、BGMゆったり |
| YouTube・ドキュメント | 長尺コンテンツ | 社員密着、エモーショナル |
| ライブ風 1分Vlog | 日常切り取り | 一発撮り風、移動中、生っぽさ |

## ワークフロー（本実装時）
1. 素材アップロード or テンプレート選択 or「AIに丸投げ」
2. Voice Profile 選択
3. AI が台本生成 → 音声合成（しゅん声クローン）→ 編集 → サムネ生成
4. 公開予約 or 即時公開（Instagram/TikTok/YouTube API）

## プロトタイプ表示要素
- KPI ボード（今月生成本数 / 総再生数 / エンゲージメント / 社員工数削減）
- Voice Profile 一覧（4種）
- 生成キュー（進捗バー付き）
- 公開済み動画ギャラリー（プラットフォーム別 / 再生・いいね表示）
- 新規作成タブ（素材UP / テンプレ / AI丸投げの3ボタン）

## 期待効果
- 動画制作: 1本3時間 → 1本15分
- 月産本数: 12本 → 42本
- リーチ: 月50万 → 月127万
