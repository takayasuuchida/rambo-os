# Module 03 — Customer Loyalty（店舗・顧客AI）

## 目的
LINE Business Connect でお客様1人1人の来店履歴・嗜好・誕生日・家族構成を学習し、
しゅんの口調で自動コミュニケーションすることで、リピート率とLTVを最大化する。

## データ統合
- POS（旬鮮四季いろは / FUCARS / K&S）
- 予約システム（試乗予約 / 内見予約 / 来店予約）
- LINE トーク履歴
- SNS フォロー状況（@shun8000 をフォローしているか等）

## AI が自動でやること
- 誕生日3日前にバースデーボトル予約のお伺いLINE
- 最終来店から14日経過したVIPに「最近どう？」LINE
- 試乗予約のリマインド（前日21:00 / 当日10:00）
- 雨予報の日、屋外席予約の方に屋根追加情報LINE
- 同店来店歴のあるお客様グループに、同時招待イベント案内

## プロトタイプ表示要素
- KPI ボード（会員数 / 再来店率 / 自動送信数 / 休眠リスク / LTV平均）
- 顧客一覧（リスクタグ：休眠/注意/健全 + LTV + 来店回数）
- 顧客詳細パネル（プロファイル + 次のアクション + LINEトーク履歴）
- LINE トーク UI（AI自動メッセージとお客様返信の対話表示）

## 期待効果
- 再来店率: 55% → 67%（+12pt）
- 休眠顧客の復活率: 8% → 24%
- 店長のLINE返信業務: 1日2時間 → 15分
