# Module 04 — Decision Support（経営判断AI）

## 目的
社員からの「これどうしたらいいですか？」電話・LINE をAIが一次受けし、
過去のしゅんの判断パターンを学習した上で「しゅんならこう判断する」を即答する。
しゅんの携帯が鳴り続ける状態を止める。

## 自動応答可能な判断レンジ
- 値引き判断（〜10%まで）
- 修理見積調整
- バイト・社員の昇給/降格（規定内）
- イベント運営の天候対応
- 仕入れ価格交渉のGOライン
- スケジュール調整
- お客様クレーム対応の方針

## 人間（しゅん本人）に強制エスカレーション
- ¥10M 超の投資判断
- 社員の解雇
- 新規事業 GO/STOP
- メディア対応・炎上対応
- 顧問弁護士マターと判定された案件
- AI 信頼度 80% 未満の判断

## プロトタイプ表示要素
- KPI ボード（本日の判断 / 平均応答 / 本人レビュー一致率 / 人間転送数）
- 判断依頼スレッド一覧（社員名 + 事業 + 緊急タグ + 質問抜粋）
- 判断詳細パネル
  - 社員の質問（QUESTION）
  - AI判断（SHUN WOULD JUDGE / 信頼度% 付き）
  - 3アクションボタン：この判断を実行 / しゅん本人に確認 / 再生成
- フッター注意書き（強制エスカレーション条件の明示）

## しゅん判断の特徴（プロンプト学習対象）
- 数字より「お客様の人生」「社員の成長」を最優先
- 規定外でも筋が通れば即決
- 損して得を取る発想（短期コスト＞長期関係）
- 横展開・クロスセル前提の判断
- 「俺が直接電話する」と本人介入を明示するパターン

## 期待効果
- しゅんへの社員電話: 1日40件 → 1日6件
- 判断スピード: 平均4時間 → 平均4.2秒
- 社員の意思決定自立度: 大幅向上（AIが先生役）
