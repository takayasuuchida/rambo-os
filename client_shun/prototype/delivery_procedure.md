## 共有フロー（5分で完了）

### Step 1. artifact 化（先輩の Mac で1回だけ）
1. `rambo-os-v0.2.jsx` を VS Code 等で開いて全文コピー（Cmd+A → Cmd+C）
2. claude.ai で新規チャットを開く
3. ペーストして、こうメッセージ：
   > 「これを React Artifact として開いて、そのまま動作確認できる状態にして」
4. Claude が artifact パネルに RAMBO OS を表示する

### Step 2. しゅん社長への共有
- artifact パネル右上の「共有」ボタンをクリック
- 共有リンクをコピー
- **LINE でしゅんに送信**（メッセージ例）：
  > しゅんさんお疲れさまです。例の経営AI、こちらで開けます。
  > Maxプラン契約済みなので追加課金ゼロでフル機能使えます。
  > [リンク]
  > iPhone の Claude アプリで開いてください。

### Step 3. しゅんが iPhone で開く
- LINE のリンクをタップ
- Claude アプリが起動 → artifact が表示される
- しゅんの Max プラン枠で API 実行（追加課金ゼロ）

## 想定 Q&A

**Q. これ俺の API キー使ってるの？**
A. いいえ。**しゅんさんの Claude Max プラン枠**で動いてます。追加課金ゼロ、API キー入力も不要です。

**Q. iPhone じゃなくて iPad でも動く？**
A. 動きます。むしろ iPad の方がデモ映えします。Mac の Claude アプリでも同じです。

**Q. 相場の数字は本物？**
A. 相場ダッシュボードの**数値はサンプルデモデータ**です。本実装フェーズで LME API・東京鉄鋼新報・日刊産業新聞のフィードを実接続します。**EVERSTEEL 88%、JETRO 7,200、関東鉄源組合 17社の数字は一次ソース検証済み**です。

**Q. 6事業全部に Voice 学習させる時間ある？**
A. はい。Phase 2 で過去の LINE・電話文字起こし・SNS 投稿を学習データに統合します。プロトタイプは「しゅん本人の判断哲学プロンプト」で動かしてます。

**Q. これ他社にも売れる？**
A. 売れます。ただし「鉄の証券会社」は絆商会の独占資産にすべきです。EVERSTEEL B2C 転用は業界初なので、しゅん社長が押さえれば**先行者利益確実**です。
